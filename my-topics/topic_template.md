## 題目名稱
基於 GitHub 的海事資訊科技學習筆記庫與數位作品集
## 為什麼對這個題目有興趣？
身為海事資訊科技系的學生，在學習 Linux、Docker 和 Git 等技術時，發現目前網路上的海事專業英文術語與資訊技術的對照資料較為零散。
我希望透過建立這個筆記庫，將課堂所學的技術文件化，並利用 Markdown 語法與 Git 版本控制系統進行管理。
這不僅能強化我的實作能力，更能透過 GitHub Pages 建立個人網站，展現我將海事專業與資訊技術整合的成果。
## 可能使用的資料來源
資料名稱來源網址資料格式課程講義與筆記學校教學平台.pdf / .txt
Linux/Git 官方文件https://git-scm.com/doc.html海事資訊專業術語表交通部航港局官網.html

## 預期成果
做出來後會是一個結構完整的 GitHub Repository，包含多個 Markdown 檔案。
透過 GitHub Pages 功能，這些筆記會轉換成一個專業的靜態教學網站，內容涵蓋 Linux 常用
指令、Git 操作流程以及海事資訊專業對照表，方便自己隨時查閱，也能分享給同學參考。
